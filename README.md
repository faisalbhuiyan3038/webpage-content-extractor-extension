# Webpage Content Extractor - Chat with any Webpage

A cross-browser extension (Chrome/Firefox) that extracts text content from any webpage and makes it easy to chat with AI assistants.

## Features

- 🚀 **One-click extraction** - Extract page content instantly
- 📝 **Custom prompts** - Create and manage your own prompts
- 🤖 **Multiple chatbots** - ChatGPT, Claude, Gemini, Grok, DeepSeek, and more
- ⚡ **Keyboard shortcuts** - User-configurable for power users
- 💾 **Import/Export** - Backup and restore your settings
- 🌙 **Dark/Light mode** - Follows system preference

## Installation

### Chrome
1. Open `chrome://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder

### Firefox
1. Open `about:debugging#/runtime/this-firefox`
2. Click **Load Temporary Add-on**
3. Select `manifest.json`

## Usage

1. Navigate to any webpage
2. Click the extension icon
3. Select a prompt and chatbot
4. Click **Extract & Copy**
5. If enabled, the chatbot opens automatically - just paste!

## Keyboard Shortcuts

Configure shortcuts in your browser:
- **Chrome:** `chrome://extensions/shortcuts`
- **Firefox:** `about:addons` → ⚙️ → Manage Extension Shortcuts

## Options

Access settings by clicking **Options** in the popup:
- **Prompts** - Add/edit/delete custom prompts
- **Chatbots** - Add/edit/delete custom chatbots
- **Shortcuts** - Instructions for setting up keyboard shortcuts
- **Backup** - Import/export your settings

## Default Prompts

- Summary - Short (under 100 words)
- 5-10 Key Points with emojis
- Summary with Key Points & Takeaways
- Blinkist-Like Summary
