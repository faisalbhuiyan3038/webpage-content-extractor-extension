How to Set Shortcuts

1

Chrome: Go to chrome://extensions/shortcuts

Firefox: Go to about:addons → ⚙️ → Manage Extension Shortcuts

2

Find "Webpage Content Extractor"

3

Set your preferred keyboard shortcuts for each action

Available Commands

Not set

Not set
